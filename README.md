# portal-qiesports-resource

## 项目简介

portal-qiesports-resource 是一个用于归档和发布独立 HTML 页面的资源仓库。所有页面均以静态 HTML 文件形式存储，不依赖特定的后端服务或数据库，适合长期保存和直接浏览。

本仓库仅用于整理与展示，不指向任何特定网站或域名。

## 目录结构

```
portal-qiesports-resource/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的公共资源文件（如图片、样式、脚本）
└── README.md       # 本说明文档
```

- **pages/**：每个子文件夹代表一个独立页面，内含对应的 HTML 文件及相关资源。
- **assets/**：全局共享的资源文件，按类型或用途组织。

## 页面归档说明

- 所有页面均为纯前端实现，可直接在浏览器中打开运行。
- 页面内容保持原始状态，不进行后期修改或美化。
- 归档遵循“一页一文件夹”原则，便于查找和管理。
- 资源文件尽量内联或随页面打包，减少外部依赖。

## 维护说明

- 欢迎提交 Issue 或 Pull Request 来补充或修正页面内容。
- 新增页面时，请在 pages/ 下创建独立文件夹，并确保 HTML 文件可自包含。
- 若引用公共资源，需在 assets/ 中统一存放，并在页面中注明引用路径。
- 请勿添加任何形式的跟踪代码、广告或外部链接。
